# AMaze v0.5

This is a terminal 2.5D game about mazes.

<div align="center"><img src="https://github.com/Melyohin-AA/AMaze/raw/master/images/main.png" alt="main"></div>

Features we have:
* 180x120 char screen resolution
* 15 char brightness levels
* 2 color palettes
* 90 degrees FOV + fisheye
* -No overdraw- Yes overdraw
* Wall collision
* Invisible walls and Ghost-walls

Features we don't have:
* Mazes
* Points of interest
* Challenges or goals

Supported platforms:
* Windows :(

Controls (press one key at a time):
* W/A/S/D - movement
* Q/E - camera rotation

Notes:
* The recommended font is Raster Font 8x9 as the most rectangular of tiny fonts.
* In legacy console mode, the "graphics" is about x1.5 times slower. Try it out.
