# AMaze v0.2

This is a terminal 2.5D game about mazes.

<div align="center"><img src="https://github.com/Melyohin-AA/AMaze/raw/master/images/corridor.png" alt="main"></div>

Features we have:
* 120x80 char screen resolution
* 7 char brightness levels
* 90 degrees FOV + fisheye
* No overdraw
* Wall collision

Features we don't have:
* Mazes
* Points of interest
* Challenges or goals

Supported platforms:
* Windows :(

Controls (press one key at a time):
* W/A/S/D - movement
* Q/E - camera rotation

Notes:
* The recommended font is Raster Font 8x9 as the most rectangular of tiny fonts.
* In legacy console mode, the "graphics" is about x1.5 times slower. Try it out.
